== Description ==

This plugin added Bangladeshi Taka (BDT &#2547;) WooCommerce plugin where WooCommerce yet not support Bangladeshi currency (BDT &#2547;) & Bangladeshi Local Payment Gateways into WooCommerce. bKash, DBBL Mobile Banking etc.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. (BDT &#2547;) will be automatically added to WooCommerce > Setting (General Options)
4. bKash & DBBl Mobile Banking added to WooCommerce > Setting > Payment Gateways
